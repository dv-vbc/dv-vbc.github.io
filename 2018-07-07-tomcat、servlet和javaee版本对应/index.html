Tomcat、Servlet和JavaEE版本对应

发表于 2018-07-07

Servlet和JSP规范版本对应关系：

Servlet规范版本 JSP版本 JSF版本 JAVA EE版本
Servlet2.3 JSP1.2、JSP1.1 J2EE1.3
Servlet2.4 JSP2.0 JSF1.1 J2EE1.4
Servlet2.5 JSP2.1 JSF1.2、JSF2.0 Java EE5
Servlet3.0 JSP2.2 Java EE6
